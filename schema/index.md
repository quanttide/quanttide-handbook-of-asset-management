# 标准资产类别

各个阶段产生的、可供交付的成果载体。它服务于特定目的和读者，形态可以是文档、数据库记录、配置文件等。交付物是工作过程的具体产出，包括日志（Journal）、札记（Essay）、手册（Handbook）、简介（Profile）、标准（Specification）等类型。

## 工作章程`bylaw`

工作章程指的是治理规则的文档，定义宗旨、权力范围和运作原则，明确边界和责任。

## 工作手册`handbook`

工作手册指的是团队协作实践的文档，针对具体岗位或业务流程的标准化指导（SOP），确保工作质量和效率。

## 工作平台`platform`

工作平台是承载组织运营、协同办公和业务流转的数字空间。
