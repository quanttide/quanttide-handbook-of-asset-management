# 第二大脑

第二大脑是组织知识管理和认知增强的数字系统，外化集体记忆与思考过程，提升组织认知能力和协作效率。

## 资产契约

第二大脑将组织知识分为陈述型记忆和程序型记忆两大类，均为九宫格结构。

### 陈述型记忆

九宫格将知识按类型维度与时间维度交织，形成九个格子。每个格子代表一种「契约」——即组织在特定时间维度上对特定类型事务所达成的共同约定。

类型维度：
- 事件类：回答「我们要做什么」
- 语义类：回答「我们相信什么知识」
- 自我类：回答「我们是谁」

时间维度：
- 过去：已履行的契约，奠定组织的连续性
- 现在：正在履行的契约，保障实时校准与执行
- 未来：将要履行的契约，提供方向约束与承诺

| | 事件类 | 语义类 | 自我类 |
|:----|:-------|:-------|:-------|
| 过去 | Archive 工作归档 | Report 工作报告 | Story 品牌故事 |
| 现在 | Journal 工作日志 | Profile 工作简介 | Brochure 宣传册 |
| 未来 | Roadmap 工作蓝图 | Context 工作语境 | Vision 工作愿景 |

### 程序型记忆

程序型记忆按约束力层次与消费对象交织，形成九宫格。

约束力层次：
- 宪法：最高约束力，规定基本原则与权责划分
- 法律：中等约束力，系统性程序规范
- 法理：低约束力，基本原理与方法论

消费对象：
- 人类友好：面向人类读者，强调可读性与启发性
- AI友好：面向AI系统，强调结构化与可解析性
- 规则引擎友好：面向自动化系统，强调可执行性

| | 人类友好 | AI友好 | 规则引擎友好 |
|:----|:---------|:-------|:-------------|
| 宪法 | Bylaw 工作章程 | Specification 工程标准 | Toolkit 工具箱 |
| 法律 | Handbook 工作手册 | Gallery 工作案例 | Platform 平台 |
| 法理 | Tutorial 工作教程 | Essay 工作札记 | Example 示例程序 |

## 资产工作流

### 陈述型记忆的未来时间维度

自上而下：
1. 具象化愿景：上级输出高感性的方向描述，确立组织的终点向量。
2. 转化语境：上下级通过深度交互，将愿景转化为具体的约束集合（优先级、资源位、边界）。
3. 制定蓝图：下级在确定的语境约束下，输出具备确定性的路径规划。

自下而上：
1. 回溯蓝图：下级从执行中的具体阻碍出发，反向检查既定路径与现实环境的摩擦点。
2. 校验语境：上下级基于冲突点重新审视原有的约束条件与优先级是否仍然有效。
3. 对齐愿景：在语境修正后，上级回到愿景确认修改后的共识仍符合最初的意图方向。
