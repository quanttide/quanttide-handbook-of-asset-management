# 标准平台资产

标准平台资产指的是一套由标准资产组成的平台资产。

标准平台资产包括：
- `src/`: 代码
- `docs/`: 文档
- `examples/`: 示例
- `scripts/`: 脚本
- `assets/`: 资源

## 代码资产

代码资产指的是用于发布的代码。

标准代码资产包括：
- `provider/`: 服务端
- `studio/`: GUI客户端
- `cli/`: CLI客户端
- `<language/framework>_sdk`: 语言或框架SDK

## 文档资产

文档资产指的是用于发布的文档。

标准平台文档资产包括：
- `brd/`: 业务需求文档
- `prd/`: 产品需求文档
- `ixd/`: 交互设计文档
- `add/`: 架构设计文档
- `dev/`: 开发者文档
- `qa/`: 质量保证文档
- `ops/`: 运维文档
- `user/`: 用户文档

## 资源资产

资源资产指的是必要的资源。

标准资源资产包括：
- `fixtures/`: 标准测试样例

## 示例资产

示例资产通常是小型项目，遵循主要语言的项目规范。

标准示例资产包括：
- `<example_name>/`: 独立的示例项目

## 脚本资产

脚本资产是以文件为单位的工具脚本。

标准脚本资产包括：
- `<script_name>.sh`: Shell 脚本
- `<script_name>.py`: Python 脚本
